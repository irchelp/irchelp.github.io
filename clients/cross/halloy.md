# Halloy

Halloy is an open-source graphical IRC client written in Rust. It aims to provide a simple and fast client for Mac, Windows, and Linux platforms.

# Features

* Graphical client
* Built with Rust
* Theming
* IRCv3.2 capabilities
* License GPL-3.0
